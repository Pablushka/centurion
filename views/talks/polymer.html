Introducción a Polymer

Crea componentes web para tus páginas y aplicaciones web con la librería Polymer de Google

Polymer Library

Polymer es una librería que te permite sacar el máximo provecho a la plataforma web a través de web components.

Esto se logra extendiendo el mismo HTML para modularizar nuestras aplicaciones web.

Web Components?

Los web components son una especificación agregada al HTML y el DOM para permitir crear widgets o componentes reusables, encapsulables e interpolables para páginas o aplicaciones web.

Web Components

  • Custom Elements: Una API para definir nuevos elementos HTML
  • Shadow DOM: Encapsulación de DOM y estilos
  • HTML Templates: Delcaración de plantillas o partes HTML
  • HTML Imports: Una manera declarativa de importar documentos HTML dentro de otros

Soporte y polyfills

Los web components son soportados en la mayoría de los navegadores actuales pero para asegurar su funcionamiento es recomendado instalar los polyfills.

Descargar

Se recomienda usar bower (el cual requiere de node.js):


$ bower install polymer

Polymer require de los web components polyfills, el cual se instala al instalarse Polymer:


$ bower install webcomponentsjs

También puedes descargarlo directamente desde el repositorio.

Instalación

En la página donde quieras instalarlo, dentro de la etiqueta <head>:


<!-- Polyfills para navegadores antiguos -->
<script src="./bower_components/webcomponentsjs/webcomponents-lite.min.js"></script>
<!-- Librería Polymer -->
<link rel="import" href="./bower_components/polymer/polymer.html">

Nuestro primer componente

Crea un archivo HTML en la misma carpeta que se llame app-user.html y dentro:


<link rel="import" href="./bower_components/polymer/polymer.html">

<dom-module id="app-user">
  <template>
    <span>Usuario:</span>
    <strong>
      <content></content>
    </strong>
  </template>

  <script>
    Polymer({
      is: 'app-user'
    });
  </script>
</dom-module>

Nuestro primer componente

En nuestra página HTML importamos nuestro componente:


<link rel="import" href="./app-user.html">

Y lo utilizamos como una etiqueta HTML:


<app-user>María Pérez</app-user>

Nuestro primer componente

Ahora demosle estilo:


<template>
  <style>
    :host {  /* El mismo elemento */
      display: block;
      margin: 0 0 1rem;
    }
    span {  /* Elementos dentro del local DOM */
      font-style: italic;
      color: blue;
    }
    :host ::content u {  /* Elementos contenidos */
      color: red;
    }
  </style>
  <span>Usuario:</span>
  <strong>
    <content></content>
  </strong>
</template>

Nuestro primer componente

Utilizando el elemento con nuevos estilos:


<app-user>
  La asombrosa <u>María Pérez</u>
</app-user>

Resultando:

Nuestro primer componente

Ahora vamos a definirle propiedades del elemento:


<script>
  Polymer({
    is: 'app-user',
    properties: {
      // Nótese que los atributos con dashes (-) se convierten
      // en camelCase (profile-img -> profileImg)
      profileImg: String,
      job: {
        type: String,
        value: 'Deportista'  // Valor por defecto
      }
    }
  });
</script>

Nuestro primer componente

Y definamos cómo utilizaremos esas propiedades:


<template>
  <style>
    /* ...estilos... */
  </style>
  <img src$="{{profileImg}}" alt="Foto de perfíl">
  <br>
  <strong>
    <content></content>
  </strong>
  <br>
  <small>{{job}}</small>
</template>

Nuestro primer componente

Utilizando el elemento con propiedades para configurar:


<app-user profile-img="user1.jpg" job="Diseñadora">
  María Pérez
</app-user>
